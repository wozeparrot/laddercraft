# Laddercraft

A minecraft server written in [zig](https://ziglang.org/).

Currently targeting 1.19.2.

## Building

```sh
zig build
```

## Running

```sh
zig build run
```

***or***

Build first then

```sh
./zig-out/bin/ladder
```

## Configuration

Change the config in `src/config.zig` then rebuild.
